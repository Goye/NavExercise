    Get paid for giving a shit.

    Technologies used: Javascript, Html, CSS, ES6

    Package manager used: Webpack and Npm.

    Css Preprocessor: SASS (packages: node-sass and loaders for webpack).

    ES6 compiler: Babel, with babily to minify the JS, CSS minification is managed by webpack.

    JSON Load: Used async await to load the data asynchronously.

    Accessibility: Aria and Wai Aria

    Unit Testing Framework: Ava

    Npm Commands:

    • Test: npm run test
    • To generate a build in public folder: npm run build
    • Obviously to run: npm start
    • to install: npm i
